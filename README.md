# Mini-Hospital-Assignment-
## Backend
  1. npm i
  2. create .env and add PORT=5050
     MONGO_URI=mongodb+srv://*name*:*password*@cluster0.th6bj.mongodb.net/mini_hms
     JWT_SECRET=change_this_secret
  3. npm run seed
  4. npm run dev
## Frontend
Vite + React + Tailwind frontend for HMS.

.env and set VITE_API_BASE_URL to your backend.

Run:
```
npm install
npm run dev
```
